# 字段角色映射表

## 角色码对照

| 角色码 | 角色名称 | 说明 |
|--------|----------|------|
| 1 | 业务主键 | 唯一标识业务实体 |
| 2 | 关联标识 | 用于关联其他表/实体 |
| 3 | 业务状态 | 表示业务状态/进程 |
| 4 | 时间字段 | 时间戳、日期相关 |
| 5 | 业务指标 | 数值型业务数据 |
| 6 | 业务特征 | 描述性业务属性 |
| 7 | 审计字段 | 创建/更新时间、用户等 |
| 8 | 技术字段 | 系统内部字段 |

## 使用场景

查询字段语义时，API 返回的 `field_role` 为数字码，需映射为中文名称展示。

```python
FIELD_ROLE_MAP = {
    1: "业务主键",
    2: "关联标识",
    3: "业务状态",
    4: "时间字段",
    5: "业务指标",
    6: "业务特征",
    7: "审计字段",
    8: "技术字段"
}
```
